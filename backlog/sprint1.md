### User Story 1  
**As a** registered user  
**I want to** reset my password if I forget it  
**So that** I can regain access to my account  

**Acceptance Criteria:**  
- A "Forgot Password" link is visible on the login page.  
- Clicking the link prompts the user to enter their registered email address.  
- An email with password reset instructions is sent to the provided email.  
- The user can successfully reset their password and log in with the new password.  

---

### User Story 2  
**As a** new user  
**I want to** sign up using my email address  
**So that** I can create an account and start using the platform  

**Acceptance Criteria:**  
- A "Sign Up" button is visible on the homepage.  
- The sign-up form requires fields for email, password, and confirmation of the password.  
- The system validates the email format and ensures the passwords match.  
- A confirmation email is sent after successful registration.  
- The user can log in after verifying their email.  

---

### User Story 3  
**As a** frequent shopper  
**I want to** save items to a wishlist  
**So that** I can easily revisit and purchase them later  

**Acceptance Criteria:**  
- A "Save to Wishlist" button is available on product pages.  
- Clicking the button adds the item to the user's wishlist.  
- The wishlist is accessible from the user’s account dashboard.  
- Items in the wishlist display their current price and availability.  
- The user can remove items from the wishlist at any time.
